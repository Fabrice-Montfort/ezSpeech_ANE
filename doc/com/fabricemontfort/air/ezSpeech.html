ezSpeech()Constructor
public function ezSpeech(enforcer:SingletonEnforcer)

This extension is based on a Singleton.

To use it, please call ezSpeech.instance();

Parameters
enforcer:SingletonEnforcer

Example
The following code shows you how to use this extension: 
         var tts:ezSpeech = ezSpeech.instance();
         
         if (tts.isSupported())
         {
             tts.debug = true;
         
             tts.setSpeed(0.6);
         
             tts.setLanguage(languages.FR);
         
             tts.say("Bonjour");
         }

setLanguage()method 
public function setLanguage(lang:String):void

This method is used to set the language for the next sentences.

If lang param is not available, switch to default languages.US

Parameters

lang:String — You can get a list of supported languages in com.fabricemontfort.ezspeech.languages


Example
The following code shows you how to use this extension: 
         tts.setLanguage(languages.FR);
         tts.say("Bonjour");
         
         tts.setLanguage(languages.US);
         tts.say("Hello");
         
         tts.setLanguage(languages.DE);
         tts.say("Hallo");
         
         tts.setLanguage(languages.ES);
         tts.say("Hola");
         
         tts.setLanguage(languages.IT);
         tts.say("Ciao");
setSpeed()method 
public function setSpeed(speed:Number):void

This method is used to set the voice speed for the next sentences.

If speed param is < 0 or is > 1, then it doesn't change the current speed

Parameters

speed:Number — Must be comprise between 0 and 1


Example
The following code shows you how to use this extension: 
         tts.setSpeed(0.2);
         tts.say("Very slow");
         
         tts.setSpeed(0.8);
         tts.say("Very speed");
